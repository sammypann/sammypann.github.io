Vega visualizations!

Shown below is the code used to create the first visualization.

{
"$schema": "https://vega.github.io/schema/vega-lite/v4.json",
"config": {
"view": {
"continuousHeight": 300,
"continuousWidth": 400
}
},
"data": {
"url": "https://vega.github.io/vega-datasets/data/cars.json"
},
"encoding": {
"color": {
"field": "Origin",
"type": "nominal"
},
"x": {
"field": "Horsepower",
"type": "quantitative"
},
"y": {
"field": "Miles_per_Gallon",
"type": "quantitative"
}
"mark": "point"
};
Visualization #1!
This data represents the obesity rate per state.
From this, we can determine the state with the highest obesity rate is Indiana, and the lowest is Colorado. (woohoo go Colorado)
